## command line

``` 
  -h, --help            show this help message and exit
  -u, --update          update config file
  -s, --search          search book
  -i BOOKID             --bookid BOOKID  download book by book id
  -n NAME               --name NAME  download book by name
  -a APP                --app APP     run as app

```
